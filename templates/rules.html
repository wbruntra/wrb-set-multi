The Rules

Set is a challenging game in which you have to find a set of three cards that match along four dimensions.

If that makes perfect sense to you, then go ahead and start playing. Otherwise, let me explain a little bit more.

The game starts by laying out a board with twelve special cards. Here are some example cards:

You'll notice that every card has four different characteristics:

color, number, shape, fill

and each of those characteristics has three different possibilities.

Three cards form a set if, for each of those four characteristics, the three cards are either all the same or all different.

That last sentence can be a little confusing, so here is an example set:

As you can see, these three cards all have the same number, same fill, different shapes, and different colors. That makes them a set.

Here is another set:

These cards all have the same shape, different number, different fill, and different colors.

This is not a set:

Even though the cards have the same shape, different numbers, and different fill, the colors are neither all the same nor all different, so there is no set.

It doesn't matter how many of the characteristics are the same, and how many are different, as long as for each characteristic, the cards are all the same or all different. Here's one last example:
